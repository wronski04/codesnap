# CodeSnap

Real-time collaborative code editor.

## Features

- Real-time collaboration via Supabase Realtime
- Debounced auto-save
- Syntax highlighting (JavaScript)
- Custom or random pad names

## Tech Stack

- Next.js 16
- TypeScript
- Supabase
- CodeMirror 6
- TailwindCSS
- Deployed on Vercel

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Create `.env.local` based on `.env.example`
4. Run: `npm run dev`
